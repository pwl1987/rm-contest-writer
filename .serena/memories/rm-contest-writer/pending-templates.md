# 待创建HTML模板清单

## P1 高优先级（Phase 1）

1. `architecture-app-innov.html` - 应用创新类流程图架构
2. `architecture-content.html` - 内容创意类传播路径架构  
3. `material-preview.html` - 材料预览页含评分卡
4. `export.html` - 导出格式选择页

## 模板内容概要

### architecture-app-innov.html
- 横向流程：选题 → 采集 → 编辑 → 审核 → 发布 → 反馈
- 系统节点：方正/索贝/融媒体平台
- 双向箭头标注数据流

### architecture-content.html
- 传播链路：内容策划 → 多平台分发 → 效果追踪 → 数据分析
- 平台节点：微信/微博/抖音/快手/客户端
- KPI标注：阅读量/点赞/转发/评论

### material-preview.html
- 左侧：Markdown渲染的申报材料
- 右侧：评分卡（通用40分 + 赛道60分）
- 每项显示分数和问题点

### export.html
- 导出格式选项：Markdown / Word / PDF
- 预览缩略图
- 确认导出按钮
