# 截图命名规则

本目录存放申报材料 Mockup 截图，按以下命名规范：

| 编号 | 文件名 | 对应模板 | 章节 |
|------|--------|---------|------|
| 01 | `scr-01-project-init.png` | project-init.html | 项目初始化向导 |
| 02 | `scr-02-analysis-confirm.png` | analysis-confirm.html | 信息采集确认 |
| 03 | `scr-03-architecture-tech-rnd.png` | architecture-tech-rnd.html | 技术研发类架构图 |
| 04 | `scr-04-architecture-app-innov.png` | architecture-app-innov.html | 应用创新类架构图 |
| 05 | `scr-05-architecture-content.png` | architecture-content.html | 内容创意类架构图 |
| 06 | `scr-06-material-preview.png` | material-preview.html | 物料预览 |
| 07 | `scr-07-export.png` | export.html | 文档导出预览 |

## 截图要求

- 格式：PNG
- 分辨率：1280×720（16:9）或原始截图比例
- 命名：`scr-##-{name}.png`
- 存放路径：`memory/templates/screenshots/`

## 引用方式

在 SKILL.md 第七步中引用：
```markdown
![项目初始化向导](memory/templates/screenshots/scr-01-project-init.png)
```
